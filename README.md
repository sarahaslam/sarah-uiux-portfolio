# Sarah Aslam UI/UX Portfolio

A quick one-page static portfolio website.

## How to publish on GitHub Pages

1. Create a new GitHub repository, for example `sarah-portfolio`.
2. Upload all files from this folder: `index.html`, `style.css`, `Sarah_Aslam_CV.pdf`, and the `assets` folder.
3. Go to repository **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/root`, then save.
6. GitHub will give you a live link after the site deploys.

## Quick edit points

- Update your LinkedIn link in `index.html` if you want it in the navigation.
- Replace images in `assets/` with better exports when you have time.
- Edit text directly in `index.html`.
